1. method
   . 실행문을 여러개 묶어서 기능을 실행
   . 결과 반환하는 경우(메서드 앞에 데이터)도 있고 반환않고 처리(void) 
2. [접근 지정자] 반환형(데이터형 또는 void) 메서드명(매개변수...) {
    구현문; 
    ...
   }
3. 매개변수(parameter)
 1) call by value
    - 매개 변수가 기본형인 경우
      전달되는 것이 값이므로 값이 변경되도 전달하기 전의 데이터는 변함없음
 2) call by reference
   - 매개 참조형인 경우(배열 또는 객체)
     주소가 전달 되므로 값이 변경되면 전달 전의 값도 같이 변경됨
4. static(정적, 공용) 메서드
  - 객체를 생성하지 않고 클래스명.메서드()
5. overloading   메서드의 오버로딩
  - 같은 클래스 내에서 메서드명이 같고 매개변수의 데이터형이 다르거나 또는
    매개변수의 갯수가 다르거나 데이터형 순서가 다른 경우
 - 같은 메서드지만 다양한 데이터형을 전달하여 처리
6. this : 인스탄스 자신의 이름, 나의 의미
   this.변수명 : 멤버변수를 가르킴
   this가 없는 변수 : 가까운 변수(지역가 있으면 지역(매개)변수) 없으면 멤버변수
     - 지역(매개)변수와 멤버변수명이 같을 때 this를 사용
